<B>Maze Game</B><br><br>

This project is an updated version of a simple maze generator based on the **DFS (Depth-First Search)** algorithm, expanded into a small maze game.<br>

<br><B>Features<B><br>
<ul type = "circle">
  <li>
    <li>Maze generation using **DFS** (from start to end and from end to start).</li>
    Player and exit cells.
    Event cells that increase the player's score.
    Locked exit that unlocks only after all event cells have been collected.
    Escape path visualization using **BFS (Breadth-First Search)** and **A*** algorithms.
    Path animation reversal.
    Difficulty level selection (each level generates a new maze and resets the current score).
    Simple graphical user interface.
    In-game background music.
  </li>
</ul><br>
