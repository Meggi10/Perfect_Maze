This project is an update version of simple generating maze using DFS (Depth-First-Search) algorithm - the simple game.
Avaliable features in application: generating maze using DFS from start to end and from end to start, added player cell and end cell (the end of the maze - to solve it), events cells that are count as score.
Work in progress: create escape path using A* or BFS (Breadth-First-Search) algorithm, adding lock exit until the player collect all points (unlock when the player collect all with appropriate score).
