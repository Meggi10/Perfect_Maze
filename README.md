This project is an updated version of a simple maze generator using the DFS (Depth-First Search) algorithm—a basic game. Available features in the application include generating a maze using DFS from start to end and from end to start, adding a player cell and an end cell (the goal of the maze to solve it), event cells that count toward the score, generating an escape path using the BFS (Breadth-First Search) algorithm, and locking the end cell (which unlocks after the player collects all points). Currently under development: implementation of the A* algorithm and a random teleport event.
