This project is an updated version of a simple maze generator using the DFS (Depth-First Search) algorithm—a basic game. <br><br>
Available features include generating a maze using DFS (from start to end and from end to start), adding a player cell and an end cell (the goal of the maze), event cells that contribute to the score, generating an escape path using the BFS (Breadth-First Search) and A* algorithms, and locking the end cell (which unlocks after the player collects all points), also added simple UI. <br><br>

<b>Currently under development:</b> improving and expanding the Game Rules section, implementing a random teleport event, composing music for the game, and creating sound effects.
